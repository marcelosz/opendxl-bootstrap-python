Installation¶

Prerequisites¶

  • Python 2.7.9 or higher in the Python 2.x series or Python 3.4.0 or higher in the Python 3.x series installed within a Windows or Linux environment.

  • The OpenDXL Python Client prerequisites must be satisfied

    • https://opendxl.github.io/opendxl-client-python/pydoc/installation.html

Installation¶

The installation files are located in the lib directory of the distribution.

Use pip to automatically install the library:

pip install dxlbootstrap-0.2.2-py2.py3-none-any.whl

Or with:

pip install dxlbootstrap-0.2.2.zip

As an alternative (without PIP), unpack the dxlbootstrap-0.2.2.zip (located in the lib folder) and run the setup script:

python setup.py install
